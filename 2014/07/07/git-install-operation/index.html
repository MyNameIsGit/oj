OJ 2.0 - Git的安装和简单操作

1. 注册一个github帐号。

2. 创建一个仓库。

3. 安装git

sudo apt-get install git

4. 创建ssh

ssh-keygen -t rsa -C "your_email"
一路按回车就行了。
完成之后，在~/.ssh文件夹中有两个文件，id_rsa和id_rsa.pub,其中id_rsa是私钥，id_rsa.pub是公钥。
把id_rsa.pub里面的东西全部复制下来，然后贴到github的帐号里。
在account settings/ssh keys/add ssh key，就可以了。

5. 测试连接。

ssh -T git@github.com
出现下面的话就证明连接上了。


# 6. 设置username , email

7. git clone

把代码从github上扒下来。
每个仓库都有个 HTTP clone URL .
直接 get clone HTTP clone URL 就可以了。
比如，git clone https://github.com/kdqzzxxcc/antry.git

8. git push

把本地代码上传。
比如我们已经把antry这个库扒下来了，现在修改了test.txt里面的东西，然后要上传。

9. Git操作还有很多，请自行百度。
