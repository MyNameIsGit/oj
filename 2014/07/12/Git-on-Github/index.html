创建SSH密钥

在Git命令行窗口中输入：
ssh-keygen -t rsa -C "your_email"
完成之后，在~/.ssh文件夹中有两个文件，id_rsa和id_rsa.pub,其中id_rsa是私钥，id_rsa.pub是公钥。
把id_rsa.pub里面的东西全部复制下来，然后贴到github的帐号里。
在account settings -> ssh keys -> add ssh key，就可以了。
测试：
ssh -T git@github.com
出现下面的话就证明连接上了。
